## Retrieval Pipeline

```
Query
  → HUGINN (Haiku): token-overlap + Jaccard pre-filter → LLM re-score
      if confidence ≥ 0.7: return result, skip MUNINN
      if confidence < 0.7: pass candidates to MUNINN
  → MUNINN (Sonnet): cosine re-rank + deep semantic rerank
  → return shard_ids with scores + reasoning
```
